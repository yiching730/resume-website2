興趣

後來發現自己真正喜歡的興趣是研究網頁設計，html運用bootstrap的grid功能切版，網頁程式設計原來很有趣，發現有趣的功能時，會覺得很好玩，如jquery的animate功能。

網頁設計是一件有趣的事情。

View my works from github

為什麼喜歡網頁設計呢？

以前工作的學校網頁使用黑快馬系統，類似網頁部落格編輯器...

後來發現html及css語法是非常有趣的，可以藉由修改語法，讓文字及功能產生不同變化，
後來又接觸到jquery、bootstrap、font awesome，發現網頁技術一直在創新，也有許多好玩的功能，讓我不斷地學習與探索。

font awesome

透過語法，讓icon圖案簡單嵌入，讓版面優化，圖片簡單明瞭。

jquery

讓網頁可以快速產生動畫功能，也可以透過語法，讓不同功能可以搭配使用。

bootstrap

透過bootstrap，讓版面配置更優化，也擴增一些功能，讓網頁功能更多，也使讀者使用上操作更快速，版面更簡潔。

CSS

透過CSS語法，讓網頁設計語法更簡潔。

HTML

網頁設計的主語法，搭配不同語言使網頁設計上更有質感。

JAVASCRIPT

透過運用JS，使網頁有一些動態功能。

自我工作期許

網頁是一件有趣的工程，從零到網頁成形，探索網頁的多樣型態

頁首、頁尾的版面配置，還有所需的功能，會發現網頁其實處處充滿探索的空間，內文的版面配置及內容也需要巧思，若一個美化又簡約的風格，能吸引到更多不同的讀者賞閱，網頁也好操作，覺得網頁設計很有趣。 希望有這個機會可以從事這個職務，除了能累積經驗外，也能培養不同的技巧，學習不同的事務，累積經驗。希望有機會能參與本職缺的面試，謝謝抽空看了履歷，真的很感謝！謝謝。
